# Ledger v22 Data Safe

גרסה זו מתקנת את שכבת שמירת הנתונים:

- טעינה אוטומטית מהענן אם קיימים פרטי Ledger מקומיים.
- מניעת דריסה של ענן קיים על ידי אתר ריק.
- Snapshot אוטומטי לפני כל שמירה לענן.
- הכנסות, הוצאות קבועות וקיזוזים נשארים חודשיים ונפרדים לפי חודש.
- תואם לפרויקט Vercel הקיים.

אחרי העלאה: פתח את האתר, עבור לגיבוי ענן, לחץ 'טען מהענן' אם הנתונים לא נטענו אוטומטית, ורק לאחר שבדקת שהנתונים מופיעים לחץ 'שמור לענן'.
